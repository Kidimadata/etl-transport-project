# ETL Transport Project

Dit project toont een eenvoudige ETL-pipeline met Python, Pandas en PostgreSQL.

Het doel van dit project is om gegevens uit een Excelbestand te halen, deze op te schonen met Python en vervolgens in een PostgreSQL-database te laden voor verdere analyse.

## ETL Pipeline

Excel  
↓  
Python (pandas)  
↓  
PostgreSQL  
↓  
SQL analyse

## Stappen van het project

1. Data extraheren uit een Excelbestand
2. Data opschonen en transformeren met Pandas
3. Data laden in PostgreSQL
4. Analyse uitvoeren met SQL

## Gebruikte technologieën

- Python
- Pandas
- SQLAlchemy
- PostgreSQL
- Excel
